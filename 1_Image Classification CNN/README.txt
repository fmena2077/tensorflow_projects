Example scripts to classify images

- simple_CNN.py: Script using a simple CNN to clasiffy black&white images

- simple_CNN_color.py: Script using a simple CNN to clasiffy color images

- ImageClassification_models.py: Load different models and see how they classify an image

- simple_CNN_color_with_TFdataaug_v20210827.py: Use TF datasets to load data, use tf.iamges to augment, and CCN with regularization

- test_surgical_tools: Model that detects surgical tools in images during surgery. NOTE: This is work in progress, model is still not completed.
